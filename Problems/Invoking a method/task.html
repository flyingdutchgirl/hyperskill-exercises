Invoking a method

There is a class named SomeClass. You need to find its only method and invoke it without arguments.

class SomeClass {

    // there is a method
}

Your program must print out the result of the found method.



Memory limit: 256 MB
Time limit: 8 seconds

Caution

You may see errors in your code or in execution results because of missing context. Don't mind those; simply write your solution and click Check.
